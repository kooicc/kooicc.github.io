## VPN客户端类型

### 1. PPTP

TCP 1723（用于PPTP连接）

### 2. L2TP/IPsec

UDP 1701（用于L2TP连接）
UDP 500（用于IKE）
UDP 4500（用于NAT穿透）

[Windows使用IPsec/L2TP客户端需要添加两个注册表](https://github.com/hwdsl2/setup-ipsec-vpn/blob/master/docs/clients-zh.md#windows-%E9%94%99%E8%AF%AF-809)

- NAT的兼容问题

  ```cmd
  REG ADD HKLM\SYSTEM\CurrentControlSet\Services\PolicyAgent /v AssumeUDPEncapsulationContextOnSendRule /t REG_DWORD /d 0x2 /f
  ```

- 启用IPsec加密

  ```cmd
  REG ADD HKLM\SYSTEM\CurrentControlSet\Services\RasMan\Parameters /v ProhibitIpSec /t REG_DWORD /d 0x0 /f
  ```

## Windows添加静态路由访问远程内网

192.168.10.0/24（远程内网地址段）到192.168.1.1（远程VPN网关）的静态路由

```bash
route -p add 192.168.10.0 mask 255.255.255.0 192.168.1.1
```
