Microsoft 激活脚本 (MAS)

开源 Windows 和 Office 激活器，具有 HWID、Ohook、TSforge、KMS38 和在线 KMS 激活

### 使用方法 1 - PowerShell在线激活 (推荐) 

1. 右键单击 Windows 开始菜单并选择 PowerShell 或终端。

```PowerShell
irm https://get.activated.win | iex
```
或者，您可以使用以下内容（将来将弃用）。

```
irm https://massgrave.dev/get | iex
```

### 方法 2 - 传统方法 (下载激活)
 
[GitHub下载链接](https://github.com/massgravel/Microsoft-Activation-Scripts/archive/refs/heads/master.zip)

[Massgrave下载链接](https://git.activated.win/massgrave/Microsoft-Activation-Scripts/archive/master.zip)

解压文件后，找到名为 `All-In-One-Version` 的文件夹，运行名为 `MAS_AIO.cmd` 的文件，您将看到激活选项，按照屏幕上的说明操作。
